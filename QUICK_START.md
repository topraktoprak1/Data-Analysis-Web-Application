## Step 1: Install Dependencies
Open PowerShell or Command Prompt in this folder and run:
```
pip install -r requirements.txt
```

## Step 2: Start the Application
Run:
```
python app.py
```

## Step 3: Open in Browser
Go to: **http://localhost:5000**

Click on "Data Analysis" in the sidebar menu.

## That's It!

### Main Features:
- Upload Excel files (.xlsx, .xls, .xlsb)
- Filter data interactively
- Create pivot tables
- Generate charts
- Save favorite reports
- Export to Excel/Word

### Need Help?
See `README.md` for detailed documentation.

### Troubleshooting:
**Q: Port 5000 is in use**
A: Edit app.py and change the port number on the last line

**Q: Missing modules**
A: Run `pip install -r requirements.txt`

**Q: Can't find templates**
A: Make sure you're running from the project folder
